\# Simple Interest Calculator



A simple, user-friendly command-line application written in Python to calculate simple interest. This project is designed to help users quickly compute the interest accrued on a principal amount over a specific period at a given annual interest rate.



\## Formula Used



The calculator uses the standard simple interest formula:



$$A = P(1 + rt)$$



Where:

\* \*\*$A$\*\* = Total Accumulated Amount (Principal + Interest)

\* \*\*$P$\*\* = Principal Amount

\* \*\*$r$\*\* = Annual Interest Rate (decimal)

\* \*\*$t$\*\* = Time Period (in years)



The isolated Simple Interest ($I$) is calculated as:

$$I = P \\times r \\times t$$



\---



\## Features



\* \*\*Quick Calculation:\*\* Computes simple interest and final balances instantly.

\* \*\*Input Validation:\*\* Ensures user inputs for principal, rate, and time are positive numerical values.

\* \*\*Clear Outputs:\*\* Displays a clear breakdown of the interest earned and the total final value.
